## How to Use This File

1. Complete manual Amazon research on 3–5 niche candidates using `templates/niche_research_template.md`
2. After each niche is scored, record observations here about patterns and gaps across niches
3. Use these notes to identify which gap represents the strongest opportunity
4. This file informs the niche selection decision — not a substitute for per-niche scorecards

**Important:** All observations must be based on personal manual Amazon browsing. No scraping tools, no automated data extraction, no services that violate Amazon's Conditions of Use. If you didn't see it yourself on Amazon.com, don't record it here.
