## How to Use This File

1. Start with this seed list
2. Search each phrase on Amazon.com (Books > Children's Books)
3. Observe what comes up — note top titles, BSRs, cover quality, review counts
4. Record observations in `research/[niche-name]/` using `templates/niche_research_template.md`
5. Add new seeds discovered through Amazon autocomplete
6. Cross off seeds that showed no demand or no gap
