## Prompt (Copy and use with Claude)

```
I'm planning the Canva layout for a children's picture book to be published on Amazon KDP under the QMN / MindQuest brand.

Canva is being used as a layout tool only. The original story, characters, and illustration concepts were developed before Canva is opened. Canva handles typography, composition, and export.

BOOK DETAILS:
- Title: [TITLE]
- Format: [PAGE COUNT]-page picture book
- Target trim size: [e.g., 8.5" x 8.5" square or 8" x 10" portrait]
- Target audience: [AGE RANGE]
- Color palette: [from approved character bible/illustration prompts]
- Tone: [e.g., warm and gentle, bright and playful]

Please help me plan:

1. TRIM AND BLEED SETUP
   - Confirm correct trim dimensions for KDP
   - Bleed setting (0.125" on all sides)
   - Safe margin for text (minimum 0.25" inside trim line)

2. TYPOGRAPHY PLAN
   - Suggest 2–3 appropriate font options for a [AGE RANGE] children's book
   - Recommended font size for story text (read-aloud use)
   - Line spacing recommendation
   - Text alignment recommendation
   - Where text should sit on a standard spread (e.g., bottom third, right-aligned)

3. PAGE LAYOUT STRUCTURES
   - Standard spread layout (text + illustration)
   - Full-bleed illustration spread (if applicable)
   - Activity page layout (if applicable)
   - Cover front layout principles for this book

4. COVER THUMBNAIL READABILITY CHECK
   - What elements must be visible and readable at approximately 160px wide
   - Typography sizing and contrast recommendations for the thumbnail

5. CANVA ASSET GUIDANCE
   - Which types of Canva elements are safe to use in a commercially published book (Pro/paid elements with commercial license)
   - What to avoid
   - How to document Canva assets used

IMPORTANT CONSTRAINTS:
- All recommendations must be compatible with KDP print specifications.
- Do not suggest Canva stock illustrations as the primary book artwork.
- Do not suggest any Canva template that would result in the book looking like a generic template-based design.
- All typography must be appropriate for the target age group.
- This plan is a draft for Elise's review — no Canva production begins until she approves it.
```
