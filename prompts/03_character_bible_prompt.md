# Prompt 03: Character Bible — QMN KDP Publishing Engine

*Use this prompt to create original characters after the book brief has been approved.*

---

## When to Use This Prompt

After:
- Book brief approved by Elise
- Proceeding to Phase 5 (Character Bible)

---

## Prompt (Copy and use with Claude)

```
I'm creating original characters for a children's picture book for Amazon KDP under the QMN / MindQuest brand.

Book title (working): [TITLE]
Target audience: [AGE RANGE]
Theme: [THEME]
Emotional arc: [BRIEF SUMMARY — from approved book brief]

Please help me develop a character bible including:

MAIN CHARACTER:
1. A working name — original, memorable, age-appropriate, not currently used by a known children's book character (I will verify this name via trademark search before finalizing)
2. Age and brief physical description — use original descriptive terms only; no comparisons to known characters
3. Core personality traits that make this character feel real and specific — not generic "nice kid"
4. The specific fear or challenge this character faces in this story
5. What makes this character lovable and relatable to a [AGE RANGE]-year-old
6. Voice notes — how this character speaks; any distinctive speech patterns

SUPPORTING CHARACTERS (suggest 1–2):
1. A supporting character (friend, family member, or mentor figure) who helps the main character
2. Name, relationship, brief personality — all original

WORLD/SETTING:
1. Primary setting appropriate for the story theme
2. Atmosphere and visual feeling

VISUAL CHARACTER NOTES (no artist references):
1. Overall visual style direction for illustrations
2. Color associations for the main character
3. What to avoid visually (to prevent IP confusion with known franchises)

IMPORTANT CONSTRAINTS:
- All character names must be original. Do not use names from known Disney, Pixar, DreamWorks, Nickelodeon, PBS, Netflix, or streaming original characters.
- Character appearances must be entirely original — no visual traits that could be confused with existing franchise characters.
- No artist names, studio names, or "in the style of" language anywhere in this output.
- All suggestions are working drafts only. Elise will review all names and descriptions before anything is finalized.
- Character names will be searched on the USPTO trademark database and Amazon before use.
```

---

## After Getting the Output

1. Copy the character concepts into `books/[book-id]/03_character_bible.md` using `templates/character_bible_template.md`
2. Search the proposed character names:
   - USPTO trademark database (uspto.gov/trademarks)
   - Amazon Books search
   - Google "[name] children's book character"
3. Document search results in the copyright check section of the character bible file
4. Submit to Elise for approval before any manuscript work begins
