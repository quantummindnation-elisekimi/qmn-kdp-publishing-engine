# Prompt 07: Copyright Check Support — QMN KDP Publishing Engine

*Use this prompt to prepare for the copyright check phase. Note: AI assists the check — Elise personally verifies and signs.*

---

## When to Use This Prompt

Before Phase 10 (Copyright/IP Review). This prompt helps Elise organize what needs to be checked. The actual searches and sign-off are done by Elise personally.

---

## Important Policy Note

This prompt generates a checklist and flags potential risks. It is a preparation tool only.

**Elise must personally:**
- Run all trademark searches (USPTO database, Amazon, Google)
- Review all illustration prompts for IP risk
- Verify all Canva asset licenses
- Complete and sign `09_copyright_check.md`

AI cannot take legal responsibility. AI can help identify what to check.

---

## Prompt (Copy and use with Claude)

```
I'm preparing a copyright and IP check for a children's picture book to be published on Amazon KDP under the QMN / MindQuest brand.

Please help me prepare a thorough pre-publication copyright checklist.

BOOK DETAILS:
- Title: [TITLE]
- Character names: [LIST ALL NAMED CHARACTERS]
- Story concept: [ONE PARAGRAPH SUMMARY]
- Illustration style direction: [BRIEF SUMMARY]
- Tools used in production: [list: Claude, Canva, image tool if any]

Please:

1. FLAG any character names that sound similar to known trademarked children's book or franchise characters — even if not an exact match
2. FLAG any story concept elements that might overlap with well-known protected works (even if the overlap seems thin)
3. IDENTIFY which character names should be searched and on which databases (USPTO, Amazon, Google)
4. REVIEW the illustration style direction for any elements that could be confused with known franchise visual styles
5. IDENTIFY what Canva asset license verification is needed
6. CONFIRM what AI disclosure is required for Amazon KDP based on the tools used
7. DRAFT the copyright check section headers for `09_copyright_check.md` so Elise can fill them in

IMPORTANT CONSTRAINTS:
- You are flagging for Elise's review — not making a legal determination.
- Do not state that anything is "definitely safe" — flag anything that needs checking.
- Elise will personally verify every item and sign the copyright check document.
- Do not suggest this check replaces legal advice for high-risk situations.
```

## After Getting the Output

1. Use the AI output to organize the check items in `books/[book-id]/09_copyright_check.md`
2. Elise personally runs all trademark and title searches
3. Elise reviews all illustration prompts
4. Elise verifies all Canva asset licenses
5. Elise signs the document

**No KDP submission without Elise's personal sign-off.**
