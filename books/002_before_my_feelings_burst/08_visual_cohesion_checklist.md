## 4. Room and Prop Continuity

**Room rules:**
- One cozy home craft-table space — warm and familiar; not clinical, not sterile
- Simple warm room — warm cream walls, warm wood floor, soft morning window light
- Minimal background details — the room has warmth and personality but is not cluttered
- Same table / drawing area across all scenes — same shape, same height, same position relative to window

**Card progression (cumulative — each state adds to the previous):**

| Stage | State |
|---|---|
| 1 | Blank / fresh card — clean white or cream, no marks |
| 2 | Pale yellow line — thin, washed-out, clearly too faint |
| 3 | Blue smudge — sweep of blue across the card face |
| 4 | Small tear — a definitive tear across the card |
| 5 | Imperfect but still Noa's — all damage visible; held with quiet acceptance |

**Prop rules:**
- Same yellow pencil — full length in spread 1, stub from spread 2 onward; never returns to full length
- Same paper / craft materials scatter on the table throughout
- Caring adult never touches the card in any spread

**Room rejection rule:** If the room changes colour, layout, or style between spreads — reject.

## 6. Hard Fail Rules

Reject immediately — no score needed — if any of the following are present:

- [ ] Noa's identity has changed — different face, hair, skin tone, or outfit
- [ ] Mom's identity has changed — different face, hair, skin tone, or outfit (any spread she appears in)
- [ ] Style has become glossy, cinematic, 3D, anime, or studio-like
- [ ] An emotion creature, blob, brain character, or monster appears
- [ ] The room has changed into a different space or a different visual world
- [ ] Card continuity has broken — wrong damage state for this spread
- [ ] Image contains embedded text, logo, or watermark
- [ ] Image is visually similar to a known book, franchise, or IP
