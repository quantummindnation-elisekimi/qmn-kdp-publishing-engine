# Niche Research Framework — QMN KDP Publishing Engine

*A 100-point scoring system for evaluating children's book niche candidates.*

---

## How to Use This Framework

After completing Phase 1–2 research (manual Amazon observation and review mining), apply this framework to each niche candidate. Score each category honestly. Use the total to decide whether to proceed.

Only niches scoring **60 or above** are viable for book concept development.

---

## Scoring Categories

### 1. Buyer Urgency (0–10)

Does this niche attract parents who feel an **urgent, immediate** need — not a vague general preference?

| Score | Description |
|---|---|
| 9–10 | Urgent, time-sensitive need (starting kindergarten, new sibling arriving, moving to a new house) |
| 7–8 | Strong ongoing need (child anxiety, fear of the dark, friendship struggles) |
| 5–6 | Moderate interest (general kindness, sharing, being nice) |
| 3–4 | Low urgency (general positivity, generic encouragement) |
| 0–2 | No clear parent pain point |

---

### 2. Search Demand Signal (0–10)

Is there evidence that parents are actively searching for books in this niche on Amazon?

| Score | Description |
|---|---|
| 9–10 | Many results, sponsored ads visible, multiple bestsellers with active BSRs |
| 7–8 | Clear demand with several titles appearing in organic search |
| 5–6 | Some titles exist but demand signal is mixed |
| 3–4 | Very few titles; unclear whether parents search this term |
| 0–2 | No results or irrelevant results |

---

### 3. Competition Weakness (0–10)

Are the top-ranking competitors beatable? Is there room for a clearly better book?

| Score | Description |
|---|---|
| 9–10 | Most top results are low-quality, poorly reviewed, or visually weak |
| 7–8 | Mixed quality — some strong competitors but clear gaps |
| 5–6 | Several decent competitors; would need strong differentiation |
| 3–4 | Strong established titles with thousands of positive reviews |
| 0–2 | Category dominated by major publishers with unbeatable authority |

---

### 4. Parent Pain Point Clarity (0–10)

Is the parent's reason for buying **specific and well-defined**?

| Score | Description |
|---|---|
| 9–10 | Very specific pain point ("my child screams every time they have to try a new food") |
| 7–8 | Clear pain point with consistent language in reviews ("my daughter was terrified of starting school") |
| 5–6 | Somewhat clear but could be more specific |
| 3–4 | Vague — "I want my child to be a good person" |
| 0–2 | No identifiable parent pain point |

---

### 5. Series Potential (0–10)

Can this niche support **3 or more books** with the same characters or theme family?

| Score | Description |
|---|---|
| 9–10 | Natural series (e.g., "The Little Brave Brain," "The Little Kind Brain," "The Little Curious Brain") |
| 7–8 | Could extend to 3–4 books with some creative effort |
| 5–6 | One or two follow-ups possible but not obvious |
| 3–4 | Feels like a one-time book with limited extension |
| 0–2 | No series potential |

---

### 6. Differentiation Potential (0–10)

Can QMN produce something **meaningfully different** from what currently exists?

| Score | Description |
|---|---|
| 9–10 | Clear opportunity for better story, stronger emotional arc, superior art direction, or unique character concept |
| 7–8 | Some differentiation possible with focused execution |
| 5–6 | Could differentiate but would require significant creative investment |
| 3–4 | Hard to stand out — existing books cover the ground well |
| 0–2 | Market is saturated with high-quality titles |

---

### 7. Production Feasibility (0–10)

Can this book be realistically produced with available resources?

| Score | Description |
|---|---|
| 9–10 | Straightforward to produce — character, setting, and illustration style are manageable |
| 7–8 | Achievable with modest creative challenge |
| 5–6 | Some complexity in illustration or story structure |
| 3–4 | High production complexity (complex characters, detailed world-building required) |
| 0–2 | Not feasible with current resources |

---

### 8. Copyright/IP Safety (0–10)

Is the niche **clearly safe** from copyright and trademark risk?

| Score | Description |
|---|---|
| 9–10 | Generic theme with no IP risk; no similar trademarked character concepts in the space |
| 7–8 | Low risk with standard due diligence |
| 5–6 | Some adjacent IP; requires careful character design and title choices |
| 3–4 | Known trademarked concepts nearby; risk of confusion |
| 0–2 | High IP risk — known characters, brands, or concepts in the space |

**Note:** A score of 3 or below on this category is a disqualifying flag. Do not proceed regardless of total score.

---

### 9. Monetization Potential (0–10)

Can this niche support **sustainable revenue** beyond a single sale?

| Score | Description |
|---|---|
| 9–10 | Strong series potential + companion products (activity book, printables, digital) + gift purchase appeal |
| 7–8 | Good repeat purchase potential or upsell path |
| 5–6 | Single-sale focused; some upsell possible |
| 3–4 | Limited to low-priced single sale |
| 0–2 | No monetization beyond base sale |

---

### 10. QMN/MindQuest Brand Fit (0–10)

Does this theme **genuinely align** with QMN's brand positioning in emotional intelligence, mindset, courage, and growth?

| Score | Description |
|---|---|
| 9–10 | Perfect fit — the book could only come from QMN/MindQuest |
| 7–8 | Strong fit with clear brand connection |
| 5–6 | Loosely aligned; would require positioning work |
| 3–4 | Tangential to QMN values |
| 0–2 | No brand fit — would dilute or confuse the QMN brand |

---

## Score Interpretation

| Total Score | Decision |
|---|---|
| **80–100** | Strong candidate — proceed to book brief with Elise approval |
| **60–79** | Revise and validate — identify weak categories, improve the angle before proceeding |
| **Below 60** | Reject or park — do not invest production resources |
| **Any category ≤ 3 in Copyright/IP Safety** | Disqualifying flag — do not proceed regardless of total |
