# Canva Design Rules — QMN KDP Publishing Engine

*Canva is a layout and design tool. It is not the source of original IP for this project.*

---

## Canva's Role in This System

Canva handles:
- Page composition and layout
- Typography (font selection, sizing, text placement)
- Trim size setup and bleed configuration
- Color palette application via Brand Kit
- Cover design layout
- KDP-compliant PDF export

Canva does **not** provide:
- Original story concepts
- Original characters
- Original illustration art that constitutes the book's IP
- Any content that QMN can claim exclusive ownership of

All creative IP — characters, story, themes, visual concepts — is developed by QMN **before** Canva is opened.

---

## Canva DO's

- **Set up correct trim sizes** for KDP (e.g., 8.5" x 8.5", 8" x 10", 6" x 9" — confirm with KDP specs)
- **Add bleed** of 0.125" on all sides when using Canva's custom dimension with bleed
- **Use Canva's Brand Kit** to apply QMN's approved fonts, colors, and logo consistently across all books
- **Use Canva for typography** — select readable, age-appropriate fonts; size text for read-aloud use
- **Use Canva for layout composition** — position where illustrations will sit relative to text
- **Use Canva for cover design** — title, subtitle, author name, back cover text layout
- **Export as PDF for Print** with correct color settings
- **Document every Canva template, stock element, or asset used** in `06_canva_notes.md` with its license type

---

## Canva DON'Ts

- **Do not use Canva stock illustrations as the primary book artwork** — Canva stock elements are not original QMN IP and may not be used as the main visual content of a commercially sold book
- **Do not use Canva "personal use only" elements** in a commercially published KDP book — check the license on every element before use
- **Do not use Canva templates as-is** without substantial original design — the book cover and interior must be meaningfully original, not a template with text swapped
- **Do not use AI-generated images from Canva's AI tools** without verifying: (a) terms allow commercial KDP publishing, (b) AI disclosure is completed on KDP
- **Do not use any Canva element that could be identified as belonging to another brand** or that resembles known characters
- **Do not rely on Canva for the original illustration concept** — illustration briefs are created first (see `05_illustration_prompts.md`), then executed, then placed in Canva

---

## QMN Brand Kit in Canva

- Create a dedicated QMN/MindQuest Brand Kit in Canva containing:
  - Primary and secondary brand colors
  - Approved heading and body fonts
  - QMN logo files (primary, secondary, icon versions)
- Apply the Brand Kit consistently across all books to build visual series coherence
- Do not mix the QMN Brand Kit with another brand kit or default Canva template palettes
- Document Brand Kit choices in `06_canva_notes.md` for every book

---

## Original Illustration Preference

**Preferred:** Original illustrations commissioned from a human illustrator.
- Human-created illustrations give QMN full IP ownership (confirm with contract)
- Avoid style similarity to known character franchises in the brief

**Acceptable with documentation:** AI-generated illustrations, provided:
1. The generation tool's terms of service explicitly permit commercial KDP publishing
2. The style was generated from original descriptions (no artist style references)
3. Amazon KDP AI content disclosure is completed
4. The generator used and the prompts are documented in `09_copyright_check.md`

**Unacceptable:** Canva stock illustrations used as primary book art; any illustration that incorporates elements from known franchises.

## Cover Thumbnail Readability

KDP shows book covers at approximately 160px wide in Amazon search results. Test every cover at small size before finalizing:

- Title text must be large enough to read at 160px wide
- Title must be high-contrast against the background (dark text on light, or light text on dark)
- Central character or image must be clear and uncluttered at small size
- Avoid small decorative text in the cover design — it will be unreadable
- The cover should attract a click at a glance

## Trim Size and Bleed Reference

KDP children's book common trim sizes:
- 8.5" x 8.5" (square — popular for picture books)
- 8" x 10" (portrait — popular for picture books)
- 6" x 9" (standard — more common for chapter books)

Bleed requirement:
- Add 0.125" (3.175mm) bleed on all edges
- Keep all critical content (text, faces, important elements) at least 0.25" (6.35mm) inside the trim line

Page count:
- Print books must have an even total page count
- KDP has minimum page count requirements — verify at time of submission

Cover:
- KDP print cover must be a single flat PDF: back cover + spine + front cover
- Spine width is calculated using KDP's spine calculator (based on page count and paper type)
- Always use the KDP spine calculator at submission time — do not guess spine width

---

## Canva to KDP Export Checklist

Before exporting from Canva for KDP:
- [ ] Trim size is correct
- [ ] Bleed of 0.125" is set
- [ ] All text is within safe margins (0.25" inside trim)
- [ ] Export setting: PDF for Print (not PDF for web)
- [ ] Fonts are embedded (Canva PDF for Print embeds fonts by default)
- [ ] Color mode is as close to CMYK as possible (Canva outputs RGB — note this; KDP will convert)
- [ ] Interior page count is even
- [ ] Cover spine width was calculated using KDP spine calculator before this layout was created
- [ ] All Canva assets used are documented in `06_canva_notes.md`
